color and font style tester β

HTMLのフォントカラーやフォントスタイルをテストするツールです。

実際にHTML上でどのように見えるのか、色やフォントの種類、スタイルなどを簡単に切り替えて確認することが出来たら便利だと思い作ってみました。

切り替えられるものは、color,background-color,font-family,font-style,font-weight,font-size,そして実際に表示する文字を書き換えることができます。

  • 対応色:web color と named color。
  • 対応フォント:Vista Ultimateにデフォルトでインストールされているフォント。
Last Modified:2009/3/29

String Modifier β

例えばブログにコードなどの特殊文字が含まれたテキストを投稿するとき、それらの文字を変換する必要がある場合がありますが、一々そんなことを自力で行うのは面倒なので、このツールで特殊文字を一発で変換します。

その他にも、prettify や Syntax Highlighter を使用している場合、タグで囲んだり class 属性や name 属性を書き込む必要がありますが、それもこのツールによって簡単に行うことができます。

Last Modified:2009/4/5

Javascript - Regular Expressions Pattern Checker β

JavaScriptの正規表現パターンのチェックを行うツールです。

正規表現の学習ツールとしての側面を持たせるため、一つ一つにサンプルと実行ボタンを設置し、すぐに動作確認が行えるようにしました。

確認可能な動作は match と replace の二つです。

基本的なパターンの他、郵便番号や携帯電話などの実際に使用可能な正規表現パターンも掲載しています。

Last Modified:2009/3/8

Keyword Search @ iTunes Store Ranking β

Google Feed API を使用して iTunes Store から Daily Ranking Feed を取得し、表示しています。

作成当初はGoogle Feed APIのデモンストレーション的なページを作るつもりでしたが、 それだけではつまらないので Google Search API を使用してランキングされたアーチスト名や曲名をクリックするだけで検索を行い、検索結果をページ右側に表示できるようにしました。

これによってランキングされた気になるアーティスト、曲名を簡単に検索することが出来るようになり、検索結果から You Tube にアップロードされたPVなどを閲覧する、などといったことが容易になります。
